FCC 题目 Title Case a Sentence

发表于 2018-09-06 | 更新于 2019-06-10 | 分类于 freeCodeCamp | 评论数： | 阅读次数：

要求

确保字符串的每个单词首字母都大写，其余部分小写。

像’the’和’of’这样的连接符同理。

当你完成不了挑战的时候，记得开大招’Read-Search-Ask’。

这是一些对你有帮助的资源:

String.split()

###源代码 

function titleCase(str){
    var arr=str.split(" ");//将传入的字符串分割为数组arr
    var arr2=[];
    var newStr;//定义一个变量用来保存最终的字符串
    for(var i=0;i<arr.length;i++){
    arr2=arr[i].split('');//将arr数组中的每一个元素再次分割为数组arr2 目的是下面找到首字母
        arr2[0]=arr2[0].toUpperCase();//将arr2中的第一个字母大写 即arr数组中的每一个元素的首字母大写
        for(var j=1;j<arr[i].length;j++){  
        arr2[j]=arr2[j].toLowerCase();//遍历arr2数组将其除了下标为0的元素全部转换成小写                        
        }                                        
        arr[i]=arr2.join("");//拼接arr2数组为arr的每一个元素                    
        newStr=arr.join(" ");//拼接arr数组为最终的字符串                
    }
    console.log(newStr);
    return newStr;
}
titleCase("I'm a litte tea pot");

解决过程没什么值得说的 略过

#####要点
join() 方法操作的对象是数组 所以是 arr.join() 而不是 arr[i].join()
